Style Sheets

Inline
Defined within the tag itself.Not reusable.
Embedded or Internal
Style is embedded within the head tags of the document. Must be copy and pasted from document to document.
Linked or External
Style is defined within a separate document. The best way to style your documents - reusable. Single file used to format multiple pages.
